## 🚀 使用方法

### 安装依赖

```bash
pip install -r requirements.txt
```

### 配置环境

编辑 `config/config.yaml` 和 `config/env/dev.yaml`

### 编写测试用例

参考 `test_data/user_module.yaml` 示例

### 运行测试

```bash
python run.py
```

### 查看报告

```bash
allure serve reports/allure
```

## ⚠️ 注意事项

1. **环境要求**：Python 3.8+
2. **依赖安装**：建议使用虚拟环境
3. **YAML语法**：必须使用2个空格缩进
4. **变量使用**：变量必须在双引号中才能替换
5. **并发测试**：确保用例之间无数据冲突
6. **敏感信息**：使用环境变量存储密码/token
